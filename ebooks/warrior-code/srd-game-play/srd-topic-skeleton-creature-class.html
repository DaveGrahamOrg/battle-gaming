Skeleton Game Creature Class

Name: Skeleton

Description: The skeletal remains of a dead corpse that moves as if it were alive

Base Class: Warrior Code Human, Warrior Code Warrior

Clothing Required: Suggested: Halloween skeleton suit.

Experience Required: As per base classes

Training Required: Referee must validate player is familiar with creature abilities and limitations.

System of Advancement:  None

Weapons Classes: All weapons with the exception of fired projectiles

Shield Classes:  Any

Armor Classes:  Any

Special Equipment Needed: Tan headband with a black circle containing a white skull.

Standard Actions: Any Sport Action, Attack Action, or Defense Action

Creature Class Actions:

Ignore Piercing Weapons – Skeletons have immunity to all piercing weapons.
