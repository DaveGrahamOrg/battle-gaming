Warrior Player Class

Name: Warrior

Description: A warrior of medieval or ancient history.  All players are by default considered to be warriors unless classified otherwise.

Base Class: None

Clothing Required: A medieval-styled tunic with belt, earthtone pants and earthtone shoes.

Experience Required: No experience needed, all new players start as warriors.

Training Required: A referee is to instruct new players on the rules of combat.

System of Advancement: None

Weapons Classes: All weapons with the exception of fired projectiles

Shield Classes:  Any

Armor Classes:  Any

Special Equipment Needed: None

Standard Actions: Any Sport Action, Attack Action, or Defense Action

Player Class Actions:  None
